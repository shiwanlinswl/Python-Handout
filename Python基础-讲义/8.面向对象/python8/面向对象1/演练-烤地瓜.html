面向对象实战1:烤地瓜

1. 目标

  • 强化 面向对象编程的设计能力，进一步理解 类、属性、方法 的构建场景 **
  • 强化 对self的理解，方法中使用self 可以获取和修改属性

2. 需求说明

烤地瓜规则：

  • 1.地瓜有自己的状态，默认是生的，地瓜可以进行烧烤
  • 2.地瓜有自己烧烤的总时间，由每次烧烤的时间累加得出
  • 3.地瓜烧烤时，需要提供本次烧烤的时间
  • 4.地瓜烧烤时，地瓜状态随着烧烤总时间的变化而改变：[0, 3) 生的、[3, 6) 半生不熟、[6, 8) 熟了、>=8 烤糊了
  • 5.输出地瓜信息时，可以显示地瓜的状态和烧烤的总时间

3. 面向对象设计

类图

3.1 地瓜类

  1. 使用 SweetPotato 类可以创建 地瓜对象
  2. 地瓜有两个属性：
    • 状态 state：字符串
    • 烧烤总时间 cooked_time：整数
  3. 定义 cook 方法, 提供参数 time 设置 本次烧烤的时间
    • 使用 本次烧烤时间 对 烧烤总时间 进行 累加
    • 根据 烧烤总时间, 设置地瓜的状态：
      • [0, 3) -> 生的
      • [3, 6) -> 半生不熟
      • [6, 8) -> 熟了
      • 大于等于8 -> 烤糊了

3.2 主程序逻辑

  1. 创建 地瓜对象
  2. 分多次 烧烤地瓜
  3. 每烧烤一次，输出地瓜信息

4. 代码实现

4.1 定义地瓜类，并且定义__init__()和__str__()方法

class SweetPotato:  # 定义类
    """地瓜类"""
    def __init__(self):
        self.cooked_state = "生的"  # 地瓜状态
        self.cooked_time = 0  # 烧烤总时间

    def __str__(self):
        return "地瓜状态:%s, 烧烤总时间:%d" % (
        self.cooked_state, self.cooked_time)

4.2 定义"烤地瓜"方法

    def cook(self, time):
        """烧烤"""
        #  本次烧烤时间累加到 烧烤总时间中
        self.cooked_time += time
        # 根据烧烤总时间 改变地瓜的状态
        if self.cooked_time >= 0 and self.cooked_time < 3:
            self.cooked_state = "生的"
        elif self.cooked_time >= 3 and self.cooked_time < 6:
            self.cooked_state = "半生不熟"
        elif self.cooked_time >= 6 and self.cooked_time < 8:
            self.cooked_state = "熟了"
        else:
            self.cooked_state = "烤糊了"
  • 对象方法中可以使用self修改、获取对象属性， 如 cook方法中使用 self 修改了 cooked_time 和 cooked_state 属性

4.3 测试代码

把上面2块代码合并为一个程序后，在代码的下面添加以下代码进行测试

# 创建对象
digua1 = SweetPotato()
digua1.cook(1)
print(digua1)  # 地瓜状态:生的, 烧烤总时间:1
digua1.cook(2)
print(digua1)  # 地瓜状态:半生不熟, 烧烤总时间:3
digua1.cook(3)
print(digua1)  # 地瓜状态:熟了, 烧烤总时间:6
digua1.cook(3)
print(digua1)  # 地瓜状态:烤糊了, 烧烤总时间:9

5. 拓展练习

5.1 烤地瓜需求拓展

  • 1.地瓜可以添加佐料,如 盐、孜然、辣酱等
  • 2.输出地瓜信息时，可以显示地瓜的状态、烧烤总时间、以及添加过的所有佐料

5.2 需求分析

  • 每个地瓜记录自己的佐料，定义属性来记录
  • 地瓜可以添加多个佐料，应该定义 容器类型属性 来记录添加的佐料
  • 地瓜类 应该 定义方法来实现添加佐料的功能

5.3 代码实现

5.3.1 添加 condiments 属性

class SweetPotato:  # 定义类
    """地瓜类"""
    def __init__(self):
        self.cooked_state = "生的"  # 地瓜状态
        self.cooked_time = 0  # 烧烤总时间
        self.condiments = []  # 佐料列表

5.3.2 定义addCondiments()方法

def add_condiment(self, condiment):
    """添加佐料"""
    self.condiments.append(condiment)

5.3.3 修改__str__()方法，在方法中使用condiments属性显示已添加的佐料


def __str__(self):
return "地瓜状态:%s, 烧烤总时间:%d, 包含的佐料：%s" % (
self.cooked_state, self.cooked_time, self.condiments)

5.3.4 再次测试

# 创建对象
digua1 = SweetPotato()
digua1.cook(1)
print(digua1)  # 地瓜状态:生的, 烧烤总时间:1, 包含佐料:[]
digua1.cook(2)
digua1.add_condiment("盐")
print(digua1)  # 地瓜状态:半生不熟, 烧烤总时间:3, 包含佐料:["盐"]
digua1.cook(3)
digua1.add_condiment("孜然")
print(digua1)  # 地瓜状态:熟了, 烧烤总时间:6, 包含佐料:["盐", "孜然"]
digua1.cook(3)
digua1.add_condiment("辣椒酱")
print(digua1)  # 地瓜状态:烤糊了, 烧烤总时间:9, 包含佐料:["盐", "孜然", "辣椒酱"]
