定义类

目标

  • 定义类
  • 定义方法

1. 定义类

定义一个类，格式如下：


class 类名:
    方法列表
  • 类名 的命名规则按照"大驼峰"

2. 定义方法

  • 类是定义对象的共同行为的，也就是说 在类中定义对象的方法
  • 定义方法:

class 类名:
    def 方法名(self): 
        ...
  • 方法的格式和函数类似，也可以设置参数和返回值，但是 需要设置第一个参数为 self
  • 注意定义方法需要在类的缩进中

demo：定义一个Car类

# 定义类
class Cat:
    # 方法
    def eat(self):
        print("猫在吃鱼....")

    def drink(self):
        print("猫在喝可乐...")
