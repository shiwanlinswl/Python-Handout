1.1 抓包分析

使用谷歌浏览器或fiddler等抓包工具完成登陆以及购票操作，进行抓包，根据 具有业务作用 或 被set-cookie 确定以下内容：

  1. url
  2. query
  3. method
  4. data
  5. Referer
  6. response.text / response.json()
