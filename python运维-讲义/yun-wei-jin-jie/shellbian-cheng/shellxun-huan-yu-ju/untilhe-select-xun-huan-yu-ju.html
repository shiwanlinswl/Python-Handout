until和select循环语句

1. until循环语句（了解即可）

until <条件表达式>
do
    命令1
    命令2
    ...
done

until循环语句的用法与while循环语句的用法类似，区别是until会在条件表达式不成立时，进入循环执行命令；条件表达式成立时，终止循环。

2. select循环语句

select循环语句的主要作用可能就是创建菜单，在执行带select循环语句的脚本时，输出会按照数字顺序的列表显示一个菜单项，并显示提示符（默认是#？），同时等待用户输入数字进行选择

select 变量名 in 菜单取值列表 
do
    命令1
    命令2
    ...
done

注意：在此结构中“in 变量取值列表”可省略，省略时相当于使用in “$@”，使用for i就相当于使用for i in “$@”

例1：用select循环打印菜单项

方法1：直接使用列表字符串

#!/bin/bash 
select name in haha heihei xixi     
do 
    echo $name 
done

执行脚本，输出：

1) haha
2) heihei
3) xixi
#? 1
haha
#? 2
heihei
#? 3
xixi
#?

方法2：把命令结果作为变量列表

#!/bin/bash 
select name in `ls /test`  
do 
    echo $name 
done

例2：调整select循环菜单项的默认提示符及利用select变量打印数字序号

#!/bin/bash
PS3="请选择："  # PS3就是控制select循环的提示符

select name in haha heihei xixi 
do 
    echo -e "你选择的是：\n $REPLY) $name"  # REPLY变量就是菜单项对应的数字
done

执行结果，输出：

1) haha
2) heihei
3) xixi
请选择：1
你选择的是：
 1) haha
请选择：2
你选择的是：
 2) heihei
请选择：3
你选择的是：
 3) xixi
请选择：

例3：打印选择菜单，按照选择一键安装不同的Web服务

示例菜单：

[root@localhost shellscript]# bash select_menu.sh
1) install lamp
2) install lnmp
3) exit
请选择：1
你选择的是：
 1) install lamp
start installing lamp...
请选择：2
你选择的是：
 2) install lnmp
start installing lnmp...
请选择：3
你选择的是：
 3) exit
exiting
[root@localhost shellscript]#

#!/bin/bash
PS3="请选择："  

select name in "install lamp" "install lnmp" "exit"
do 
    echo -e "你选择的是：\n $REPLY) $name"
    case "$REPLY" in
        1)
            echo "start installing lamp..."
            ;;
        2)
            echo "start installing lnmp..."
            ;;
        3)
            echo "exiting"
            exit
            ;;
        *)
            echo "你输入的数字有误："
    esac
done
