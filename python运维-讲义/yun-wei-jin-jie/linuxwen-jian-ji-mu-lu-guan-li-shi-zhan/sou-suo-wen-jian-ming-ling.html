搜索文件命令

1. which：显示命令的全路径

命令作用：

which命令用于显示命令的全路径，我们常用这个命令来查找命令在哪里，which命令查找的范围是PATH环境变量的路径。

语法格式：

which [选项] [命令名]

例1：查找pwd命令存放在哪里

[root@localhost ~]# which pwd
/usr/bin/pwd

2. whereis：显示命令及其相关文件全路径

命令作用：

whereis命令用于指定命令的可执行文件、源码文件及man帮助文件的路径。

语法格式：

whereis [选项] [文件名]

常用选项：

  • -b：查找可执行文件
  • -m：查找man帮助文件

例1：将相关的文件都查找出来。

[root@localhost ~]# whereis cp
cp: /usr/bin/cp /usr/share/man/man1/cp.1.gz

# 只查找cp可执行文件
[root@localhost ~]# whereis -b cp
cp: /usr/bin/cp

# 只查找cp的帮助文件
[root@localhost ~]# whereis -m cp
cp: /usr/share/man/man1/cp.1.gz

3. locate：快速定位文件路径

命令作用：

locate命令查找文件时是直接查找mlocate.db文件，这个文件包含系统文件的文件名及对应的路径信息。使用locate命令时需要先用updatedb命令更新一下mlocate.db数据库文件。

例1：使用locate命令查找python.txt文件

# 第一次使用locate命令提示没有那个文件或目录
[root@localhost ~]# locate python.txt
locate: 无法执行 stat () `/var/lib/mlocate/mlocate.db': 没有那个文件或目录

# 需要更新mlocate.db数据库
[root@localhost ~]# updatedb

# 这个时候就可以查找出结果
[root@localhost ~]# locate python.txt
/root/python.txt

注意：使用locate命令时，如果提示“command not found”，执行yum -y install mlocate命令

4. find：查找目录下的文件

命令作用：

find命令用于查找目录下的文件，同时也可以调用其他命令执行相应的操作。

语法格式：

find [路径] [选项] [action]

常用选项：

  • -name：按照文件名查找文件，只支持*、?、[]等特殊通配符

  • -type：查找某一类型的文件：

    • b：块设备文件

    • c：字符设备文件

    • d：目录

    • p：管道文件

    • l：符号链接文件

    • f：普通文件

    • s：socket文件

    • D：door

  • -maxdepth levels：查找的最大目录级数，levels为正整数

  • -mtime [-n|n|+n]：按照文件的修改时间来查找文件（这个参数最常用），具体如下：

    • -n ：表示文件更改时间距现在n天以内
    • +n：表示文件更改时间距现在n天以前
    • n：是距现在第n天

例1：查找passwd这个文件

[root@localhost ~]# find / -name passwd
/sys/fs/selinux/class/passwd
/sys/fs/selinux/class/passwd/perms/passwd
/etc/pam.d/passwd
/etc/passwd
/usr/bin/passwd
