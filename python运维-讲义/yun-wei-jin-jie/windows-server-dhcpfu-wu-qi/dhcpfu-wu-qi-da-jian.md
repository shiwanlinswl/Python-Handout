# DHCP服务器搭建

# 1. 需求

某公司一共有150个员工，每个人都有自己的办公电脑，每个电脑都需要分配一个内网IP地址，如果采用手工配置，会很不方便，这个时候需要搭建一台DHCP服务器，**网段为：192.168.10.0/24**
