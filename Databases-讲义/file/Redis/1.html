Redis 环境安装

当前Ubuntu虚拟机中已经安装好了redis，以下步骤可以跳过

卸载旧版本的Redis（低于3.2.x版本）

1. 卸载软件
python@ubuntu:~$ sudo apt-get remove redis-server

2. 清除配置
python@ubuntu:~$ sudo apt-get remove --purge redis-server

3. 删除残留文件（一般残留文件如下）
python@ubuntu:~$ sudo find / -name redis

python@ubuntu:~$ sudo rm -rf var/lib/redis/
python@ubuntu:~$ sudo rm -rf /var/log/redis
python@ubuntu:~$ sudo rm -rf /etc/redis/
python@ubuntu:~$ sudo rm -rf /usr/bin/redis-*
python@ubuntu:~$ sudo rm -rf /usr/local/bin/redis-*
python@ubuntu:~$ sudo rm -rf /usr/local/redis/

安装

下载地址：http://redis.io/download

  1. 下载安装文件，或者通过 wget 下载

    python@ubuntu:~$ wget http://download.redis.io/releases/redis-3.2.8.tar.gz

  2. 解压

    python@ubuntu:~$ tar -zxvf redis-3.2.8.tar.gz

  3. copy文件并ls查看

    python@ubuntu:~$ sudo mkdir -p /usr/local/redis/
python@ubuntu:~$ sudo cp -r redis-3.2.8/* /usr/local/redis/
python@ubuntu:~$ ls /usr/local/redis/

  4. 进入安装目录

    python@ubuntu:~$ cd /usr/local/redis/

  5. 编译

    首先打开README.md，翻阅基本build和install方式。

       python@ubuntu:~$ vi README.md
   python@ubuntu:~$ sudo make

    尝试环境是否可以正常使用(Hint: It's a good idea to run 'make test' ;)

       python@ubuntu:~$ sudo make test

    如果出现 \o/ All tests passed without errors!，则表示redis环境 没有问题。

  6. 安装

    最后安装路径

    python@ubuntu:~$ cd src
python@ubuntu:~$ sudo make install

  7. 查看编译好的命令文件

    python@ubuntu:~$ ls /usr/local/bin/redis-*
--/usr/local/bin/redis-benchmark     # 性能测试工具,例如：redis-benchmark -n 1000000 -c 50 , 50个客户端，并发1000000个SETs/GETs查询
--/usr/local/bin/redis-check-aof     # 更新日志检查
--/usr/local/bin/redis-check-rdb     # 本地数据文件检查
--/usr/local/bin/redis-cli           # 命令行操作工具
--/usr/local/bin/redis-server        # 服务器程序

  8. 修改配置文件位置（默认配置文件目录为/usr/local/redis/redis.conf）

    python@ubuntu:~$ cd ..
python@ubuntu:~$ sudo mkdir /etc/redis
python@ubuntu:~$ sudo cp /usr/local/redis/redis.conf /etc/redis/

启动redis

  python@ubuntu:~$ sudo redis-server /etc/redis/redis.conf

出现以下代表启动成功

连接redis服务器

新开一个窗口，或者redis-server后台启动。

通过

python@ubuntu:~$ redis-cli

得到

127.0.0.1:6379>

测试：

127.0.0.1:6379> ping
PONG

代表redis服务器已经正常安装并且可以使用了。
