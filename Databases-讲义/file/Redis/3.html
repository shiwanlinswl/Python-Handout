Python数据分析课程讲义

服务器端

  • 服务器端启动Redis的命令为redis-server
  • 可以使用help查看帮助文档
python@ubuntu:~$ redis-server --help
  • 启动
# 使用指定配置文件，启动redis-server
python@ubuntu:~$ sudo redis-server /etc/redis/redis.conf
  • 停止
# 1. 在启动服务shell界面 Control+C 停止服务

# 2. 因为Redis可以妥善处理信号，所以可以直接kill -9
python@ubuntu:~$ ps -aux | grep 6479
python@ubuntu:~$ sudo kill -9 PID号

# 3. 在redis-cli客户端里 执行 shutdown命令
127.0.0.1:6379> shutdown

客户端

  • 客户端启动redis shell的命令为redis-cli
  • 可以使用help查看帮助文档
python@ubuntu:~$ redis-cli --help
  • 运行测试命令，远程连接则加 -h 192.168.xxx.xxx
python@ubuntu:~$ redis-cli 
127.0.0.1:6379> ping
PONG
  • 切换数据库
  • 数据库没有名称，默认有16个，通过0-15来标识
  • 如果没有切换，默认在第 0 个数据库里
127.0.0.1:6379> select 1
OK
127.0.0.1:6379[1]> select 0
OK
127.0.0.1:6379>

Redis 可视化客户端工具

Redis自带一个最小化的命令行式的数据库管理工具，有时侯使用起来并不方便。

Redis Desktop Manager，是一款基于Qt5的跨平台Redis桌面管理软件

  • 支持: Windows 7+, Mac OS X 10.10+, Ubuntu 14+
  • 特点： C++ 编写，响应迅速，性能好。但不支持数据库备份与恢复。
  • 项目地址： https://github.com/uglide/RedisDesktopManager
  • 下载地址与安装方式：https://redisdesktop.com/download
